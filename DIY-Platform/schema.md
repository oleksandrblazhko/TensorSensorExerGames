### Підключення одного 3-провідного тензодатчика до HX711
Для 3-провідного датчика A− обовʼязково з’єднується з E− (перемичка).


        3-wire Load Cell                 HX711
      ┌─────────────────┐        ┌───────────────────┐
Red ──┤ E+              ├──────▶ │ E+               │
Blk ──┤ E-              ├──────▶ │ E- ◀─────┐      │
Wht ──┤ A (Signal)      ├──────▶ │ A+       │       │
      └─────────────────┘        │ A- ◀─────┘       │
                                 │                  │
                                 └──────────────────┘
