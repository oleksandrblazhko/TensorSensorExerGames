## Загальна архітектура (логічна схема)
```text
┌───────────────────────────────────────────────┐
│           ПЛАТФОРМА БАЛАНСУ (TOP VIEW)        │
│                                               │
│  LC1 (FL) ─────┐        ┌───── LC2 (FR)       │
│                │        │                     │
│                │        │                     │
│  LC3 (BL) ─────┘        └───── LC4 (BR)       │
│                                               │
└───────────────────────────────────────────────┘
       │            │            │            │
     HX711#1     HX711#2       HX711#3       HX711#4
       │            │            │            │
       └────────────┴────────────┴────────────┘
                    │
              Arduino Nano
```

## Підключення одного 3-провідного тензодатчика до HX711
Для 3-провідного датчика lj (A-) необхідно підключити через міст з двами резисторами 1к:
- ((E-) -> (1k)) * ((E+) -> (1k)) -> (A-)
- https://youtu.be/LIuf2egMioA?t=511
- https://www.youtube.com/watch?v=43xcnJxuvjY
- https://github.com/Picaio/Hx711
  
```text
        3-wire Load Cell              HX711
      ┌─────────────────┐        ┌───────────┐
Red ──┤ A+              ├──────▶ │ E+        │
Blk ──┤ E-              ├──────▶ │ E-        │
Wht ──┤ E+              ├──────▶ │ A+        │
      └─────────────────┘        │ A-        │
                                 │           │
                                 └───────────┘
```
## Схема підключення HX711 до Arduino Nano
```text
┌────────────────────────────────────────────────┐
│ Arduino Nano                                   │
│                                                │
│ D2  ◀── HX711 #1 DT     D3 ◀── HX711 #1 SCK   │
│ D4  ◀── HX711 #2 DT     D5 ◀── HX711 #2 SCK   │
│ D6  ◀── HX711 #3 DT     D7 ◀── HX711 #3 SCK   │
│ D8  ◀── HX711 #4 DT     D9 ◀── HX711 #4 SCK   │
│                                                │
│ 5V  ─────────▶ VCC всіх HX711                 │
│ GND ─────────▶ GND всіх HX711                 │
└────────────────────────────────────────────────┘
```
